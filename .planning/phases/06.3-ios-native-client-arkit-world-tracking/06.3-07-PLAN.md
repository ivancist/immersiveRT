<objective>
Build the hidden 2-finger corner long-press gesture recognizer (D-12) as a self-contained UIKit component with unit-tested hit-test geometry. Per RESEARCH.md Pattern 3, SwiftUI has no primitive for "N touches, each constrained to a screen region, held for a duration," so this is a custom `UIGestureRecognizer` subclass wrapped for SwiftUI. The reveal must be very hard to trigger accidentally because the full screen is already the touch-capture surface (D-04/D-12).

Purpose: Provide the reveal mechanism for the hidden Recenter/Disconnect menu (wired in Plan 08) without interfering with normal touch capture.
Output: `CornerLongPressRecognizer.swift` (+ SwiftUI wrapper) and pure hit-test region tests.
</objective>
